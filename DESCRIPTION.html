Provide documentation for Python numeric literal.
为 Python 数字字面值提供文档。

Usage／用法

Just move your mouse over a literal integer. Bits view will be shown instead of Radix view if you hold down Ctrl in the same time.

将鼠标移到整数字面值上方即可显示不同进制下的形式，若同时按住 Ctrl 键，则改为显示比特位视图。

Plugin supports documentation for all numeric literals, but currently only integer have actual functionality.

注意：插件支持对所有数字字面值提供悬浮文档，但目前仅整数有实际功能。
